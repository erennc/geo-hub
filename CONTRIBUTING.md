# Contributing to GEO Hub

Thanks for your interest in contributing! All contributions are welcome.

## How to Contribute

### Add a Tool
1. Fork the repo
2. Edit the Awesome List section in `README.md`
3. Open a PR with the tool name, URL, and a short description

### Add a Guide
1. Fork the repo
2. Create a new `.md` file in `/docs`
3. Link it in the Guides section of `README.md`
4. Open a PR

### Add a Template
1. Fork the repo
2. Add your template to `/templates`
3. Open a PR

### Fix Something
Just open a PR — no issue required for small fixes.

## Guidelines
- Keep descriptions concise (one sentence per tool/resource)
- Use existing formatting as a reference
- Only add tools/resources you've personally used or verified
- No affiliate links

## Code of Conduct
Be respectful. Be helpful. Be constructive.
