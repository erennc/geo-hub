# GEO for E-commerce

> How to optimize your e-commerce site for AI-powered search engines.

## Why GEO Matters for E-commerce

When users ask AI engines "What's the best [product] for [use case]?", you want your products to be recommended. GEO helps you become the cited source.

## Key Strategies

### 1. Product Schema Markup
```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "Product Name",
  "description": "Clear, factual description",
  "brand": { "@type": "Brand", "name": "Brand" },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.5",
    "reviewCount": "120"
  }
}
```

### 2. Comparison Content
- Create honest product comparison pages
- Include specific specs and data points
- Compare against known alternatives

### 3. Review & Rating Signals
- Encourage verified customer reviews
- Display aggregate ratings
- Respond to reviews publicly

### 4. FAQ Pages Per Product Category
- Answer common purchase questions
- Include pricing, shipping, compatibility info
- Use FAQ schema markup

## Quick Wins
1. Add Product schema to all product pages
2. Create a "Best [Category] for [Use Case]" guide
3. Add FAQ schema to category pages
4. Include comparison tables with real data
