# 📄 GEO Templates

Ready-to-use templates for immediate GEO implementation.

## Available Templates

| Template | File | Description |
|----------|------|-------------|
| `llms.txt` | [llms.txt](./llms.txt) | Standard format for AI engine discovery |
| FAQ Schema | [schema-faq.json](./schema-faq.json) | JSON-LD FAQ structured data |
| Organization Schema | [schema-organization.json](./schema-organization.json) | JSON-LD Organization markup |

## How to Use

1. Copy the template you need
2. Replace placeholder values with your actual data
3. Deploy to your website

### `llms.txt`
Place at the root of your website: `https://yoursite.com/llms.txt`

### Schema.org Templates
Add to the `<head>` section of your HTML pages inside a `<script type="application/ld+json">` tag.
